### **条件表达式 `cond_expression` 的代码解释**

#### **1. 条件表达式的语法**
```python
x = true_value if condition else false_value
```

- **`condition`**：一个布尔表达式，用来判断条件是否成立。
- **`true_value`**：如果 `condition` 为真时的返回值。
- **`false_value`**：如果 `condition` 为假时的返回值。

#### **适用场景**：
- 适用于简单的单行逻辑，方便简化代码。
- 不适合嵌套复杂逻辑，否则会降低代码可读性。

---

### **2. 强制规范示例**

#### **YES：单行简单条件表达式**
对于简单逻辑，条件表达式可以提高代码简洁性：
```python
# 根据条件决定变量值
age = 18
status = "adult" if age >= 18 else "minor"
print(status)  # 输出 "adult"
```

等价于：
```python
# 使用 if-else 语句
age = 18
if age >= 18:
    status = "adult"
else:
    status = "minor"
print(status)  # 输出 "adult"
```

---

#### **NO：嵌套条件表达式**
嵌套条件表达式可能让代码难以阅读和维护。

**错误示例**：
```python
# 嵌套条件表达式（降低可读性）
t = 1200
unit = "seconds" if t < 60 else "minutes" if t < 3600 else "hours"
print(unit)  # 输出 "minutes"
```

**改进后的写法**：
将嵌套逻辑改为 `if-elif-else` 结构，提升可读性：
```python
# 使用 if-elif-else 改写
t = 1200
if t < 60:
    unit = "seconds"
elif t < 3600:
    unit = "minutes"
else:
    unit = "hours"
print(unit)  # 输出 "minutes"
```

---

### **3. 条件表达式的优点与局限**

#### **优点**：
- 提高代码简洁性，尤其在一行中完成简单条件判断时。
- 易于理解的小型逻辑适合使用。

#### **局限**：
- 嵌套或复杂条件时，可读性显著降低。
- 对团队协作来说，过于复杂的表达式会导致维护困难，尤其对于不同语言背景的开发者。

---

### **4. 示例对比**

#### **示例 1：简化逻辑判断**
**推荐**：
```python
# 判断分数是否及格
score = 75
result = "Pass" if score >= 60 else "Fail"
print(result)  # 输出 "Pass"
```

**等价的 if-else 写法**：
```python
score = 75
if score >= 60:
    result = "Pass"
else:
    result = "Fail"
print(result)  # 输出 "Pass"
```

---

#### **示例 2：嵌套条件的改进**
**不推荐嵌套条件表达式**：
```python
# 嵌套条件表达式（不推荐）
t = 50
unit = "seconds" if t < 60 else "minutes" if t < 3600 else "hours"
```

**推荐用 if-elif-else**：
```python
# 改写为多分支
if t < 60:
    unit = "seconds"
elif t < 3600:
    unit = "minutes"
else:
    unit = "hours"
```

### **5. 总结规范**

| **规范**                     | **正例**                                                     | **反例**                                                   |
|------------------------------|------------------------------------------------------------|----------------------------------------------------------|
| 条件表达式仅用于简单逻辑        | `x = true_value if cond else false_value`                  | 嵌套条件表达式：`x = a if cond1 else b if cond2 else c`    |
| 多条件分支用 if-elif-else 代替 | 多分支：`if cond1: ... elif cond2: ... else: ...`          | 多层嵌套：`result = a if cond1 else b if cond2 else c`     |

**规则**：**条件表达式限于简单逻辑，复杂逻辑应改用 `if-elif-else`。**
